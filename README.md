This is my programming project for a school assignment. 

To use this program you will need to have openSCAD on your PC.


Download the .scad file

Open openSCAD 

inside openSCAD

if you do not have the Customizer window on the right side of the program navigate to the top bar and inside the drop down menu for "Window" find the button saying "Customizer" and press it and you should see the window pop up on the right side.

In the Customizer bar press the parameter Arrow

Change any value to whatever you want and make your own custom shelf either for 3D printing or other manufacturing.



The rounded corner feature does not quite work yet. 


Use parameter window to customize shelf to your liking. 
All parameters are based on each shelf size.
